Usage Example

<!-- Include Lucide -->
<script src="https://unpkg.com/lucide@latest"></script>

<!-- Use icons -->
<i data-lucide="heart" class="icon-lg icon-primary"></i>

<!-- Initialize Lucide -->
<script>
  lucide.createIcons();
</script>

How to Use unDraw

1

Browse

Visit undraw.co and search for illustrations that match your content

2

Customize Color

Use the color picker to set the accent color to #88C7A5 (our mint green)

3

Download SVG

Download the customized SVG and add it to your project

Pro Tip: Always set the primary color to #88C7A5 before downloading to maintain brand consistency across all illustrations.
